# ملخص نظام جيش المسوقين - منصة Dasmai

## نظرة عامة
نظام شراكة وعمولة للمسوقين الميدانيين لدعم منصة Dasmai لبيع السيارات

## نظام الرتب (Tiers)

### 1. الرتبة البرونزية (Scout - المستكشف)
- **الحصول عليها:** مباشرة عند التسجيل
- **الصلاحيات:** جلب عملاء عبر رابط إحالة، إضافة بيانات سيارات
- **العمولة:** 10% من عمولة المنصة
- **القيود:** لا يحق له القيام بمهام ميدانية

### 2. الرتبة الفضية (Verifier - الموثق)
- **الحصول عليها:** بعد جلب 10 سيارات بنجاح + تقييم عالي
- **الصلاحيات:** مهام تصوير السيارات + المعاينة الأولية
- **العمولة:** النسبة الأساسية + مبلغ مقطوع عن كل مهمة

### 3. الرتبة الذهبية (Ambassador - السفير)
- **الحصول عليها:** 50 عملية ناجحة + اجتياز اختبار
- **الصلاحيات:** فحص فني معتمد، تمثيل المنصة في نقل الملكية، إدارة نزاعات البيع
- **العمولة:** 20% من عمولة المنصة + مكافآت ربع سنوية

## مكونات Dashboard المطلوبة

### 1. إدارة الهوية والوصول
- واجهة تسجيل خاصة بالشركاء
- التوثيق الإلزامي عبر نفاذ

### 2. نظام إدارة الإحالات والعروض
- منشئ العروض (Listing Creator)
- نظام الربط (Lead Tracking) - كود فريد لكل شريك
- أرشفة البيانات

### 3. محرك العمولات الذكي
- حساب المقاصة التلقائي
- المحفظة الرقمية (3 حالات: قيد التحقق، معلق للضمان، قابل للسحب)
- سجل العمليات (Transaction History)

### 4. وحدة العمليات الميدانية
- نظام المهام (Task Ticketing)
- إثبات الحضور (GPS)

### 5. الحماية والحوكمة
- نظام التقييم التبادلي
- كاشف التلاعب (Anti-Fraud)
- إدارة العقود الرقمية

### 6. محفزات الأداء (Gamification)
- شريط التقدم (Progress Bar)
- لوحة الشرف (Leaderboard)
- شارات الإنجاز (Badges)

## التقنيات المقترحة
- Backend: Node.js أو Python (Django)
- Frontend: React Native
- Cloud: AWS
- Maps: Google Maps API
- Database: PostgreSQL
